Fracture Architecture

Local AI video scene splitter · PyQt6 · FFmpeg · CLIP · DBSCAN

Presentation

  • • PyQt6 dark/light desktop UI
  • • Media Pool grid of scene thumbs
  • • Drag-reorder timeline + delete
  • • Hover preview via OpenCV thread

Logic

  • • AnalysisWorker: detect → cluster
  • • ExportWorker: stitch selections
  • • All heavy work off the UI thread
  • • CLIP model loaded once & cached

System / Media

  • • ffprobe I-frames = fast scene cuts
  • • CLIP embeddings = visual meaning
  • • DBSCAN groups similar scenes
  • • FFmpeg concat = lossless export

Fracture · VideoClassifier · local-only · no cloud required
